# Utilisation

1. Si vous utilisez Google Colab, ce que je ne vous conseille pas car les entraînements sont longs, alors vous pouvez utilisez **Main-RemoteVersion.ipynb**,
2. Mais si vous avez décidé de cloner le présent répertoire GitHub sur votre machine personnelle, vous pouvez utilisez **Main-LocalVersion.ipynb**.

# Concernant la version distante

1. Connecter-vous sur Google Colab : https://colab.research.google.com,
2. Rendez-vous dans l'onglet GitHub et inscrivez l'URL suivante dans la barre de recherche : https://github.com/EnzoN7/Image-classification.git,
3. Sélectionner le fichier **Main-RemoteVersion.ipynb**.
